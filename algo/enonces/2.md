# Algorithme : Addition

Objectif: Demander deux nombres en entrée, renvoyer le résultat de leur addition. 

VARIABLES: 
* a (premier opérande)
* b (second opérande)
* resultat (le résultat)

```
DEBUT
   Demander un nombre à ranger dans a
   Demander un nombre à ranger dans b
   resultat = a+b
   Renvoyer resultat   
FIN
```
